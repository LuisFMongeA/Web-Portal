Todo Portal

A clean, fast personal task manager built with modern web technologies.

About the Application

MyTodo Portal is a personal task management web application that allows authenticated users to create, organize, and track their daily tasks. Each user has their own private todo list, accessible only after a valid login.

Features

  • Secure email-based authentication with JWT tokens
  • Create, edit, and delete personal todo items
  • Mark todos as completed with a single click
  • Persistent session across page refreshes

Technology Stack

  • Frontend: Angular 20+, TypeScript, Signals, Reactive Forms
  • Backend: .NET Web API, In-memory cache
  • Architecture: Standalone components, feature-based folder structure, HTTP interceptors

Design Principles

This application was built following modern Angular best practices — standalone components, signal-based state management, smart/dumb component separation, and a clean layered architecture designed to scale.
